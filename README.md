# Performance-test
Este es un sistema de Gestión de Tareas con Persistencia de Datos que permite:
• Registrar nuevas tareas.
• Consultar la lista de tareas.
• Buscar una tarea por algún criterio (ej. ID, título o estado).
• Actualizar la información de una tarea.
• Eliminar tareas. 
• Guardar las tareas en csv.
• Cargar tareas de un archivo csv.
